It is likely that exotel is allowing only certain IP addresses to access the mp3 files. If you are a Exotel heavy user, they won’t allow recorded mp3 files available from any IP address as it may expose user sensitive data.

You need to setup a separate server to download files from Exotel and put it in a separate cloud storage, and put those urls in your CRM.
