chrish October 13, 2025, 7:45pm 6

Hey hey.

If anyone is having issues with this still, try adding extra_hosts to your docker compose file, so it looks something like this.

  frappe:
    image: docker.io/frappe/bench:latest
    platform: linux/amd64
    command: sleep infinity
    environment:
      - SHELL=/bin/bash
    volumes:
      - ..:/workspace:cached
      # Enable if you require git cloning
      # - ${HOME}/.ssh:/home/frappe/.ssh
    working_dir: /workspace/development
    ports:
      - 8025:8025
      - 9020:9020
    extra_hosts:
      - "development.localhost:host-gateway"
      - "development2.localhost:host-gateway"

I was seeing errors like this:

44/development.localhost,{"message":"Unauthorized: Error: getaddrinfo ENOTFOUND development.localhost"}
