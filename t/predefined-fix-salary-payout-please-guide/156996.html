You could do it with a minimum of one salary structure which would be assigned 80 times (each employee must be assigned a salary structure). The Salary Structure Assignment is where you can set unique values for ‘base’ and ‘variable’. If these two fields don’t provide enough uniqueness, you may need additional Salary Structures.

How to use ‘base’ and ‘variable’ fields in Salary Structure Assignment:

base (can be an hourly/daily/weekly/monthly rate.

variable (can be used as bonus/added income/rate for retirement 401k contributions/etc)

Cases when you may need additional Salary Structures:

  1. You have hourly and salaried employees
  2. Some employees get paid daily/weekly while others get paid monthly
  3. Some employees have unique income like commissions or deductions like child support (even these might be shared in a structure if “only print if not zero” is set in the Salary Component)
  4. Some employee’s salaries are assigned to a different costing account.
