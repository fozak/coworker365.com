ERPNext, HRMS & Frappe Framework v16 Release Dates

Announcements
release, hr, erpnext, frappe
ankush November 26, 2025, 6:52am 53

Update on the release date. We have decided to delay the stable release considering the following factors:

  • Feedback on beta release + missing polish of new UI changes, especially the new navigation and desktop view.
  • New addition in scope - We are making UI similar to Espresso-based new apps.
  • Ongoing security audit of v16 code changes
  • Pending docs updates
  • Code cleanups, deprecation and migrations

New plan:

  • We’ll continue releasing beta versions as and when major milestones are achieved. Expect 3-4 more beta releases. These can be breaking in nature, so read the accompanying release notes and migration guide before updating.
  • We’ll do a “release candidate” mid-December when we’ve frozen all major changes. Only fixes will be merged after this freeze.
  • Final release will be on 12th January 2026. (~3 weeks after RC release, after intense internal, partner and community-driven testing and fixes)

I know it sucks to wait and delay the release like this, but it’s better to do it right the first time. Let’s (actually) make this the most stable major release so far.
