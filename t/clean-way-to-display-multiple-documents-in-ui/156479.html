Easily Redirect Users to Filtered List View When Multiple Docs Are Need to Show

When a workflow or action creates multiple documents, the message dialog usually shows links for each document. This becomes lengthy and hard to read â€” and once you click any link, the message disappears.

To solve this, you can now show one clean link that redirects the user to a filtered List View containing only the newly created documents 

from frappe.utils import get_filtered_list_url, get_filtered_list_link

Example

docnames = [
    "MFG-WO-2025-00027",
    "MFG-WO-2025-00028",
    "MFG-WO-2025-00029",
]

# Get Filtered List URL
url = get_filtered_list_url("Work Order", docnames)

# Get HTML link (for show_alert, msgprint, etc.)
link = get_filtered_list_link("Work Order", docnames, label="View Work Orders")

This will show one simple link like:

View Work Orders

Clicking it opens a List View filtered to only those records

Clean UI, no long messy dialogs.

Documentation: Utility Functions

Note: Available in 15.88.0 of frappe.
