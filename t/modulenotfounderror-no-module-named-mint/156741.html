Either a custom app or module called “mint” is referenced in your site configuration or database, but not installed locally, or an import path in your ERPNext code or hooks is pointing to a module that doesn’t exist in your local setup.

Check which apps are installed using “bench --site list-apps”
Check for references to “mint” in site config or hooks
