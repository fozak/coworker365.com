Hi @gdhelp03

Incase your Sign Board FG item is unique each item requires recording phase either one of the following;

1. Use a dynamic BOM for each order

  1. Copy the existing BOM
  2. Adjust raw materials according to the current requirement
  3. Use that BOM in the Work Order.

2. Use a single BOM but adjust raw materials directly while creating the Work Order

  1. Create Work Order
  2. Before submitting, open the Materials tab
  3. Add or remove items according to actual requirement
  4. Submit WO
  5. Manufacture as normal

3. Add Additional Materials in the Stock Entry (Manufacture)

  1. Record stock entry of Manufacture typee
  2. Add extra raw materials manually
  3. Submit

Manufacturing settings needs to be set up for extra raw materials to be consumed.
