Ok, I missunderstood your requirement.
Use formatters for customize listview ‚Ä¶

frappe.listview_settings['Item'] = {
    formatters: {
        image(value) {
            if (value) {
                return `<img src="${value}" style="height: 40px; width: auto; border-radius: 4px;" />`;
            }
            return '';
        }
    }
};
