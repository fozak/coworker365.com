Open Source

Like all Frappe applications, Frappe Cloud is open source. Our code is freely accessible on GitHub. Unlike proprietary software, were not a black box and you can inspect the code we run to verify everything we tell. One of the top reasons for enterprises to adopt open-source software is security and the ability to own your data. Like Linux, more contributors means more eyes, more inspection, and better security.

ISO 27001; GDPR compliant

We are committed to preserving confidentiality, integrity, availability, and privacy. Frappe's Quality and Information Security Systems are certified under the ISO 9001:2015 and 27001:2013 standards by globally leading ISOQAR. Additionally Frappe Cloud is also compliant with GDPR compliances.

Access Control and Logs

Frappe Cloud is equipped with granular access controls which can be enabled at site, bench or server level. Every database or application activity is logged for audit and analysis.

Virtual Private Cloud

The proxy server ensures restricted access to the application servers. With clusters, you also get an isolated virtual network with VPN controls.

Managed security patches

Frappe Cloud's underlying infra, OS, and technologies like database go through regular upgrades and are at par with the latest security updates

Internal security policies

Frappe deploys internal security measures such as responsible disclosures, regular hardening, and periodic VAPT assessments to detect and fix vulnerabilities.

Powered by AWS and OCI

Frappe Cloud servers are built on top of AWS and OCI, hence all the security benefits (eg. DDOS) are extended to Frappe Cloud.

Application security for ERPNext and Frappe Framework

Frappe apps (like ERPNext) are equipped with OTP/2FA, IP restriction, granular field-level roles and permissions, and an advanced audit trail, providing an added layer of security.

SSL and HTTP/2

Frappe Cloud provides free SSL certificates for all your domains. It also serves your sites on the cutting edge HTTP/2 protocol.
